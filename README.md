# graph2tikz

GUI上で直感的にグラフを描画し、論文やスライド用のTikZコードをリアルタイム生成するWebアプリケーション。

**🌐 App URL:** https://replica0820.github.io/graph2tikz/

## 🎯 Goal
グラフ理論の学習や、研究・論文執筆における作図の負担を軽減し、思考を妨げないシームレスなツールを提供することを目指しています。

## 📖 使い方 (How to use)

キャンバス（左側の白いエリア）上で以下の操作が可能です。

* **頂点の作成:** キャンバスの空いている部分をクリック
* **頂点の移動:** 頂点をドラッグ＆ドロップ
* **辺の作成:** 1. 始点となる頂点を `Shift + クリック` （頂点が青くハイライトされます）
  2. 終点となる頂点を通常クリック
* **頂点の複数選択:** 頂点を `Alt + クリック` （複数選んだ状態で上部の「横に揃える」などのボタンが使えます）
* **辺の曲がり具合の調整:** 辺をクリックして選択し、上部メニューに出現するスライダーを動かす
* **要素の削除:** 頂点や辺を `右クリック`

右側のパネルにはリアルタイムでTikZコードが生成されます。「Copy」ボタンを押すことで、そのままLaTeXのドキュメントに貼り付けることができます。

## ✨ Current Features (現在の機能)
現状はフロントエンドのみで完結しており、ブラウザ上で以下の機能が利用可能です。

- **直感的なグラフ配置:** キャンバス上のクリック操作で頂点と辺（有向/無向）を自由に配置・移動。
- **柔軟なエッジ調整:** スライダーを用いた辺の曲がり具合（カーブ）の個別調整。
- **スマートな整列機能:** 選択した複数の頂点を、水平・垂直・円状にワンクリックで美しく整列。
- **TikZコードのリアルタイム生成:** - 描画したグラフの座標を適切なスケール（cm単位）に自動変換。
  - 左から右、上から下へと人間が読みやすい順序で頂点番号を自動ソートして出力。
  - ワンクリックでクリップボードへコピー可能。

## 🚀 Roadmap (実装予定機能)

### Phase 1: 描画・スタイルの強化
- [ ] **グリッド吸着（スナップ機能）**
  - 頂点の配置や移動時、一定のグリッド間隔に自動で座標を丸め、より正確な作図をサポート。
- [ ] **辺のスタイル変更（点線・波線）**
  - 長いパスの省略表現などに用いる波線（TikZの `decoration=snake` 相当）や点線の描画。
- [ ] **マッチングの表現（辺のハイライト）**
  - 辺の背後に太い半透明の線を描画し、特定のマッチングや重要なパスを視覚的に強調。
- [ ] **要素のカラー変更と太さ調整**
  - 頂点・辺の色指定、およびスライダーによる辺の太さ変更機能（TikZ出力に連動）。

### Phase 2: グラフ理論・アノテーション向けの拡張
- [ ] **独立したテキストラベル（フリーノード）**
  - キャンバス上の自由な位置に、数式やテキスト（例: $M_2(v)$ や $+$ 記号など）を配置できる機能。
- [ ] **自己ループの描画**
  - 始点と終点が同一の頂点になる辺の作成（TikZの `loop` オプション連携）。
- [ ] **弧状配置ツール**
  - 既存の「円状配置」に加え、指定した角度範囲で頂点を弧の形に自動整列する機能。
- [ ] **Undo / Redo（元に戻す・やり直し）**
  - 操作ミスを瞬時に取り消せる履歴管理機能。

### Phase 3: 高度なグループ化と背景レイヤー
- [ ] **頂点集合の自動バウンディングボックス**
  - 選択した複数の頂点を包み込むように、自動で計算された余白付きの枠（四角形や角丸）を描画。
- [ ] **サイクル表現用の背景円**
  - 円状に配置された頂点の背後に、グラフの構造（大きなサイクルなど）を示すガイドラインとしての円を描画。

### Phase 4: バックエンド連携・エクスポート強化
- [ ] **TikZコードからのグラフ逆生成**
  - 既存のTikZコードを読み込み、GUI上のオブジェクトとして復元する機能。
- [ ] **透過画像（PNG/SVG）の直接エクスポート**
  - FastAPIバックエンドやSVG変換を利用した高品質な画像保存。
